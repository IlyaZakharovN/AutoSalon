# AutoSalon

Проект выполняется в рамках ВКР.
Цель проекта - разработка веб-приложения для автоматизации деятельности сотрудников автосалона.

Бекэнд - Django + PostgreSQL
Фронтенд - React + Redux Tooltik + Redux Persist
